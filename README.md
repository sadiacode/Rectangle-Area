# Rectangle Area
## Description
This project is a simple python program that calculates the area of rectangle.
## Technologies Used
- Pycharm
- Python 3
## How to run
1. Clone the repository.
2. Open the project in pycharm.
3. Run the 'main.py' file.
4. Enter length and width.
5. View the result of calculation.
## Sample Output
```Area
Enter length: 5
Enter width: 3
Area is 15.0
```
